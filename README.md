# Personal Portfolio

A modern developer portfolio built to showcase my projects, skills, and experience.  
This website highlights my work, provides ways to get in touch, and demonstrates my frontend and fullstack development capabilities.

## Features

- Responsive modern UI
- Projects showcase
- Contact form with email integration
- Smooth navigation
- Dark mode support
- Clean and minimal design

## Tech Stack

**Frontend**
- Next.js
- React
- TypeScript
- Tailwind CSS

**Backend / API**
- Next.js API Routes

**Email Service**
- Resend

**Icons**
- Lucide React

**Deployment**
- Vercel
